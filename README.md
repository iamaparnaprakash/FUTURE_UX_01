FUTURE_UX_01

## 📌 Overview
Future_UX_01 is a complete UI/UX website design created using Figma.  
It represents a modern, clean, and user-friendly service-based website with structured navigation and multiple pages.

The design focuses on usability, clarity, and conversion-focused user experience.

---

## 🧩 Pages Designed

- 🏠 Home Page  
- 🛠️ Services / Categories Page  
- ℹ️ About Page  
- 📩 Contact Page  
- 🛒 Cart Page  
- 👤 My Account Page  

---

## 🎯 Project Objective

To create a clean and intuitive website UI that:
- Improves user experience
- Provides smooth navigation between pages
- Encourages user engagement and conversions
- Follows modern UI/UX design principles

---

## 🧠 UX Design Approach

- Clean and minimal interface design  
- Clear navigation bar across all pages  
- Strong visual hierarchy (headings, text, buttons)  
- Consistent spacing and alignment  
- Proper use of call-to-action buttons (Order Now / Explore Menu)  
- User-friendly layout structure  

---

## 🛒 Cart Page (Empty State UX)

The Cart page is designed with an **empty state message**, indicating when no items are added.  
It includes a clear call-to-action button (“Explore Menu”) that redirects users to the Services/Categories page.

This improves user experience by:
- Avoiding dead-end pages  
- Guiding users toward actions  
- Maintaining smooth navigation flow
